38 · GPU Programming with nvexec

Standard Section: — stdexec Reference: nvexec/, examples/nvexec/

Overview

nvexec is NVIDIA’s GPU backend, wrapping the CUDA programming model as a std::execution scheduler.

cpp
namespace nvexec {
  class stream_scheduler;

  // Single GPU
  class stream_context;
  stream_scheduler stream_context::get_scheduler();

  // Multi GPU
  class multi_gpu_stream_scheduler;
  class multi_gpu_context;
}

Basic Usage

cpp
#include <nvexec/stream_context.cuh>

nvexec::stream_context ctx;
auto gpu_sch = ctx.get_scheduler();

// Execute on GPU
auto result = sync_wait(
  starts_on(gpu_sch,
    just(42)
      | then([] __device__(int x) { return x + 1; })
  )
);
// result = 43

GPU Algorithms

nvexec provides GPU-optimized algorithm implementations:

bulk

cpp
starts_on(gpu_sch,
  just(device_data)
    | nvexec::bulk(1024, [] __device__(int idx, auto& data) {
        data[idx] = data[idx] * 2;
      })
);

reduce

cpp
starts_on(gpu_sch,
  just(device_data)
    | nvexec::reduce([] __device__(int a, int b) { return a + b; })
);

when_all

cpp
starts_on(gpu_sch,
  when_all(
    gpu_kernel_1(),
    gpu_kernel_2(),
    gpu_kernel_3()
  )
);

Multi-GPU Support

cpp
nvexec::multi_gpu_context ctx;
auto multi_gpu_sch = ctx.get_scheduler();

// Execute in parallel on all GPUs
starts_on(multi_gpu_sch,
  when_all(
    gpu_work(0),  // GPU 0
    gpu_work(1),  // GPU 1
    gpu_work(2)   // GPU 2
  )
);

CUDA Stream Integration

Each stream_scheduler is associated with a CUDA stream:

cpp
// Default CUDA stream
auto gpu_sch = ctx.get_scheduler();

// Pass external CUDA stream
cudaStream_t my_stream;
// nvexec supports custom stream configuration

Domain Mechanism & GPU

The GPU scheduler uses the domain mechanism to replace standard algorithms with GPU-optimized versions:

cpp
// Standard when_all
when_all(gpu_work1, gpu_work2);
// Under nvexec domain:
// → Automatically replaced with GPU version of when_all (using CUDA events for synchronization)

Build Requirements

cmake
# CMake configuration
cmake -S . -B build -DSTDEXEC_ENABLE_CUDA=ON
cmake --build build

Requires NVIDIA HPC SDK (nvc++) or a CUDA-capable Clang.

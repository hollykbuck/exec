How to Use

Each tutorial follows the same structure:

  1. Standard Section Reference: Corresponding section numbers from exec.tex
  2. stdexec Reference: Corresponding source files and examples
  3. Core Concepts: Clear explanations and definitions
  4. Code Examples: Directly compilable code snippets
  5. Key Takeaways: Summary tips

It is recommended to read in order, but you can jump to topics of interest if you are already familiar with some concepts.

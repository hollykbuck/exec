C++26 std::execution Tutorial Series

On this page

A systematic introduction to the C++26 std::execution Sender/Receiver model from scratch.

It is recommended to read in order, but you can jump to topics of interest if you are already familiar with some concepts.

Each chapter has runnable test code at https://github.com/hollykbuck/execcode/. I use GitHub Actions to verify every test compiles and passes on Windows, Linux, and macOS. Don’t tell me this code doesn’t work on my machine :), it works on Microsoft’s machine.

Feel free to send feedback, suggestions, and discussion to https://github.com/hollykbuck/exec/discussions! Email is also welcome at slam_prance536#simplelogin.com.

Status quo:

Since C++26 std::execution is not yet implemented by mainstream compiler standard libraries, you need stdexec to try out the feature. Since Conan has not accepted stdexec in conan center index, you can install stdexec locally using the same method as my CI: link

# optional: git clone https://github.com/NVIDIA/stdexec.git
# I made some changes to conanfile in stdexec
git clone https://github.com/hollykbuck/stdexec.git
# optional: git checkout 086ccb28f85dcd7103247b6b56db540083e3a556
cd stdexec
conan export .
